# Sales Report Dashboard

## Overview
This project presents an interactive **Sales Analytics Dashboard** that provides insights into revenue trends, customer purchasing behavior, and product performance.

The dashboard helps businesses analyze:
- Total revenue
- Average order value
- Order delivery time
- Product and category performance
- Monthly sales trends
- Top ordering cities

## Key Metrics
- Total Revenue: ₹35,20,984
- Total Orders: 1000
- Average Amount Spent: ₹3,520.98
- Average Delivery Time: 5.53 days

## Dashboard Insights
The dashboard includes the following visualizations:

- Revenue by Product
- Revenue by Category
- Revenue by Time (Hourly)
- Revenue by Month
- Top Cities by Orders
- Occasion-based Sales Analysis

## Tools Used
- Power BI
- Excel / CSV Dataset
- Data Visualization
